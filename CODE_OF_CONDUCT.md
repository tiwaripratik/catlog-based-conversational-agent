# Contributor Covenant Code of Conduct

Our community is dedicated to providing a welcoming and supportive environment for all contributors. We follow the Contributor Covenant v2.1.

Please be respectful, considerate, and collaborative. If you experience or witness unacceptable behavior, contact the project maintainers.

For the full text and reporting details, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
